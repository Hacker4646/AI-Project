#  AI Agency – PowerPoint Slide Generator

##  Project Overview
This project is an AI-powered tool that automatically generates professional PowerPoint presentations based on the user's topic or input text. The system makes creating presentations faster, easier, and more efficient — perfect for students, teachers, businesses, and agencies.

---

##  Objectives
- Automatically generate presentation slides using AI  
- Save time in creating PowerPoint decks  
- Convert user-entered text or topic into structured PPT slides  
- Provide consistent formatting and layout

---

## Features
- Enter a topic or paste your own text  
- AI generates:
  - Title slide  
  - Content slides  
  - Bullet points  
  - Summary slide
- Auto-creates a `.pptx` PowerPoint file
- Professional-looking slide formatting

---

## Technologies Used
- Python  
- python-pptx (PowerPoint file creation)  
- OpenAI / GPT model or any other LLM  
- Flask / Streamlit (optional UI)
